GitHubPages是什么？

GitHubPages 是 GitHub 公司为 github 上的工程所提供的概览网页。如果将上传的一个工程比作一本书，那么 GitHubPages 可以看做书本的封面。有封面的目的很简单，就是为了使工程看起来更美观和更容易读懂。
